## 🧩 OOP Design Structure

Each algorithm is divided into two classes:

* **Logic Class** → Contains sorting logic
* **Demo/Main Class** → Handles user input and execution

Example:

```
BubbleSort.java       → Sorting Logic
BubbleSortDemo.java   → Main Execution
```

This follows proper **Object-Oriented Programming principles**.

---

## ⚙️ How to Compile and Run

### Compile

```
javac BubbleSort.java BubbleSortDemo.java
```

### Run

```
java BubbleSortDemo
```

Same process applies to:

* InsertionSortDemo
* SelectionSortDemo
